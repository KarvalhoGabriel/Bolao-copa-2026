🔥 Goleada — +4 gols de diferença

📌 Exemplos
Resultado real: Japão 4 × 0 Bósnia
Você colocou 2×0→25 PTS ✓
Você colocou 1×0→25 PTS ✓

🎯 Placar Exato Triunfo
Acertou o placar exato de uma partida com vencedor.
📌 Exemplos
Resultado real: Brasil 2 × 1 Argentina
Você colocou 2×1→20 PTS ✓
Você colocou 3×2→15 PTS (mesmo saldo)
Você colocou 2×0→10 PTS (só o vencedor)
20PTS
🤝 Empate Exato
Acertou o placar exato de um jogo que terminou empatado.
📌 Exemplos
Resultado real: EUA 1 × 1 Irã
Você colocou 1×1→20 PTS ✓
Você colocou 2×2→15 PTS (empate, mas não exato)
15PTS
✅ Vencedor + Diferença de Gols
Acertou quem ganhou e o saldo de gols, mas o placar foi diferente.
📌 Exemplos
Resultado real: Bélgica 2 × 1 Iraque
Você colocou 3×2→15 PTS ✓ (saldo +1, igual)
Você colocou 1×0→15 PTS ✓ (saldo +1, igual)
15PTS
🤝 Empate sem Ser Exato
Previu empate, o jogo empatou, mas o placar foi diferente.
📌 Exemplos
Resultado real: Japão 0 × 0 Suécia
Você colocou 1×1→15 PTS ✓
Você colocou 2×2→15 PTS ✓
10PTS
👍 Apenas o Vencedor
Acertou quem ganhou, mas errou o saldo de gols.
📌 Exemplos
Resultado real: Espanha 3 × 1 França
Você colocou 2×0→10 PTS ✓ (saldo diferente)
Você colocou 1×0→10 PTS ✓
0PTS

📌 Exemplos
Resultado real: Brasil 2 × 0 Croácia
Você colocou 0×1→0 PTS ✗
Você colocou 0×0→0 PTS ✗ (marcou empate mas houve vencedor)
